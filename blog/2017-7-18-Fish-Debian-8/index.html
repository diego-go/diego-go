Instalar Fish en Debian

July 18, 2017   

Se deberán agregar los siguientes repositorios con el siguiente comando

echo 'deb http://download.opensuse.org/repositories/shells:/fish:/release:/2/Debian_8.0/ /' > /etc/apt/sources.list.d/fish.list

Descargamos la llave del repositorio para después agregarla

wget http://download.opensuse.org/repositories/shells:fish:release:2/Debian_8.0/Release.key

apt-key add - < Release.key

Actualizamos los paquetes e instalamos Fish

apt-get update apt-get install fish

Instalando oh-my-fish

oh-myfish es una extensión que hace a fish más bonito y le da un mejor toque a la terminal para que sea más productiva. Para instalarlo e instalarle cosas, en este caso temas, hay que hacer lo siguiente.

Nos descargamos oh-my-fish con el siguiente comando e iniciará la instalación

curl -L https://get.oh-my.fish | fish

Uso de oh-my-fish

oh-my-fish trae consigo una pequeña utilidad donde se pueden hacer varias cosas, incluyendo actualizar la extensión e instalar paquetes nuevos.

Para actualizar la misma extensión

omf update [paquete a actualizar]

Instalar paquetes

omf install [nombre del paquete a instalar]

Para listar los paquetes que tenemoss instalados

omf list

Para aplicar un tema

omf theme [nombre del tema descargado]

Remove un paquete

omf remove [nombre del paquete a remover]

Remover fish :‘-(

omf destroy

Esto es lo básico para usar, para más info y pasos su web oficial

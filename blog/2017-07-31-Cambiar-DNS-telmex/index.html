Cambiar DNS Telmex

August 17, 2017   

Abrir una terminal y poner el siguiente comando para checar la dirección de nuestro gateway

route -n

Después entramos por telnet a esa dirección

telnet 192.168.1.254

Se desplegará un login donde solicitaŕa usuario y password con el que se accesa al modem vía web. Una vez dentro, desplegamos la lista de los dns que están actualmente configurados para copiarlos de alguna forma por si llegaramos a necesitarlos después.

dns server forward list

Después se vacía la lista para borrar los dns

dns server forward flush

Para configurar los DNS, utilizamos el siguiente comando, poniendo en dns= la dirección DNS que querramos usar

dns server forward dnsset add set=0 dns=8.8.4.4 metric=10 intf=Inter net owner=PPP dns server forward dnsset add set=0 dns=8.8.8.8 metric=10 intf=Inter net owner=PPP

Al hacer esto, verificamos que estén en la lista los nuevos DNS con el comando de antes dns server forward list

Y finalmente guardamos los cambios con saveall
